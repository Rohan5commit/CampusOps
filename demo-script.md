# Demo Script (6 minutes)
1. Landing page (30s): Explain student ops pain and CampusOps AI value.
2. Workspace overview (45s): Show KPI cards including urgent, overdue, completed.
3. Task board (60s): Add a task live, move status across columns.
4. Meetings AI (90s): Paste notes, run summarize, show decisions/risks/action items, then click “Add action items to Task Board”.
5. Knowledge Q&A (75s): Ask sponsor strategy question and point out citations.
6. Deadlines view (30s): Show timeline updates from added action items.
7. Closing (30s): Highlight fallback reliability + scale roadmap.
